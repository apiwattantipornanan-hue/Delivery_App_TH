# Delivery App TH

Customer ordering demo for a Thai fresh spring roll home business.

## Current Demo

Open this file in a browser:

```text
customer-ui-demo.html
```

The demo includes:

- Customer product selection screen.
- Multi-product slider for `ปอเปี๊ยะสด`, `เต้าหู้`, and `กุนเชียง`.
- Separate quantities per product.
- Cart summary.
- PromptPay QR payment mockup.
- Visual direction based on Pattern 01: fresh, warm, LINE-native, and practical.
